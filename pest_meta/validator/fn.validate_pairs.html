Expand description

It checks the parsed grammar for common mistakes:

  • using Rust keywords
  • using Pest keywords
  • duplicate rules
  • undefined rules

It returns a Result with a Vec of Errors if any of the above is found. If no errors are found, it returns the vector of names of used builtin rules.
